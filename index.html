Promocode for Skybox

Enter your promo. Note that it should be eight-digit.

Promo code rules!

The input field accepts an eight-digit code consisting of paired digits, which can be:

  • - both are even
  • - both odd
  • - even and odd

If a pair of even digits is separated by a pair of odd digits, such a promotional code has a cost of UAH 1000.

If a pair of even digits is separated by a pair of odd digits, and at the same time, in pairs of even numbers, the numbers are in ascending order (for example, 24ХХ68ХХ), such a promotional code has a cost of 2000 UAH.

If the previous conditions are not met, then we calculate the sums of even and odd digits. If the sum of the even ones is greater than the sum of the odd ones, such a promotional code has a cost of 200 UAH.

If the sum of the odd ones is greater than the sum of the even ones, such a promotional code has a cost of UAH 100.

Any other combinations return a value about incorrect promotional code.

*By the condition of the problem, when implementing the work of the promotional code, you cannot use the string data type.
